# What is the Model Context Protocol (MCP)?

## 개요

MCP (Model Context Protocol)는 AI 애플리케이션을 외부 시스템에 연결하기 위한 오픈소스 표준입니다.

MCP를 사용하면 Claude나 ChatGPT와 같은 AI 애플리케이션이 다음과 같은 것들에 연결할 수 있습니다:

- **데이터 소스** (예: 로컬 파일, 데이터베이스)
- **도구** (예: 검색 엔진, 계산기)
- **워크플로우** (예: 특화된 프롬프트)

이를 통해 AI가 핵심 정보에 접근하고 작업을 수행할 수 있게 됩니다.

> **💡 비유**: MCP는 AI 애플리케이션을 위한 USB-C 포트와 같습니다. USB-C가 전자 기기를 연결하는 표준화된 방법을 제공하듯이, MCP는 AI 애플리케이션을 외부 시스템에 연결하는 표준화된 방법을 제공합니다.

---

## MCP로 무엇을 할 수 있나요?

- **개인화된 AI 비서**: 에이전트가 Google Calendar와 Notion에 접근하여 더 개인화된 AI 비서로 작동
- **웹 앱 생성**: Claude Code가 Figma 디자인을 사용하여 전체 웹 앱을 생성
- **기업 데이터 분석**: 기업 챗봇이 조직 전체의 여러 데이터베이스에 연결하여 채팅을 통한 데이터 분석 지원
- **3D 디자인 및 프린팅**: AI 모델이 Blender에서 3D 디자인을 만들고 3D 프린터로 출력

---

## MCP가 중요한 이유

MCP는 생태계 내 위치에 따라 다양한 이점을 제공합니다:

### 개발자

MCP는 AI 애플리케이션이나 에이전트를 구축하거나 통합할 때 개발 시간과 복잡성을 줄여줍니다.

### AI 애플리케이션 또는 에이전트

MCP는 기능을 향상시키고 최종 사용자 경험을 개선할 데이터 소스, 도구 및 앱의 생태계에 대한 접근을 제공합니다.

### 최종 사용자

MCP는 필요할 때 데이터에 접근하고 사용자를 대신하여 작업을 수행할 수 있는 더 강력한 AI 애플리케이션이나 에이전트를 제공합니다.
